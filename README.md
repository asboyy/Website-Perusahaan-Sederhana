

# Website Perusahaan Sederhana 🏢

Proyek ini adalah sebuah profil perusahaan (*company profile*) berbasis web yang dirancang dengan antarmuka bersih, modern, dan informatif. Website ini bertujuan untuk memberikan informasi fundamental mengenai identitas perusahaan, layanan yang ditawarkan, serta cara menghubungi tim profesional di baliknya.

## 🌟 Fitur Utama

* **Hero Section:** Halaman muka yang impresif untuk menyambut pengunjung.
* **About Us:** Penjelasan mendalam mengenai visi, misi, dan latar belakang perusahaan.
* **Our Services:** Daftar layanan atau produk unggulan yang disediakan.
* **Contact Information:** Detail kontak dan formulir agar klien potensial dapat terhubung dengan mudah.
* **Responsive Design:** Tampilan yang tetap rapi dan fungsional di perangkat mobile maupun desktop.

## 🛠️ Teknologi yang Digunakan

Website ini dibangun dengan fokus pada performa dan kemudahan pemeliharaan menggunakan:

* **HTML5:** Untuk struktur konten yang semantik dan SEO-friendly.
* **CSS3:** Untuk styling kustom, penggunaan Flexbox/Grid, serta animasi transisi yang halus.
* **JavaScript:** Untuk memberikan interaktivitas pada elemen navigasi atau komponen dinamis lainnya.

## 📂 Struktur Repositori

```text
Website-Perusahaan-Sederhana/
│
├── index.html        # Struktur utama halaman web
├── style.css         # File stylesheet untuk desain visual
├── script.js        # Logika interaktif sisi klien
└── assets/           # Media pendukung (Gambar, Ikon, Logo)

```

## 🚀 Cara Menjalankan Secara Lokal

1. **Clone repositori:**
```bash
git clone https://github.com/asboyy/Website-Perusahaan-Sederhana.git

```


2. Masuk ke direktori proyek.
3. Buka file `index.html` menggunakan browser pilihan Anda (Chrome, Edge, atau Firefox).
